# 2023-summer-fullstack-workshop

## Slides

```bash
npm i -g @slidev/cli
```

## Backend

- `/backend`: sample code on slides

  ```bash
  cd backend
  poetry run uvicorn backend.main:app --reload
  ```

- `/pratice`: demo code in class

  ```bash
  cd pratice
  poetry run uvicorn pratice.main:api --reload
  ```
